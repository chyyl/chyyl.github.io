一、免费 MCP 全景图

1.1 MCP 是什么

MCP（Model Context Protocol，模型上下文协议）是 Anthropic 于 2024 年底开源的协议，用于让 AI 模型与外部工具、数据源、服务进行标准化通信。你可以把它理解成”AI 界的 USB 接口”——不管什么模型，不管什么工具，插上就能用。

核心概念：

MCP 定义了三种能力：

  • Tools：可调用的函数（如”执行 SQL 查询”）
  • Resources：可读取的数据源（如”当前 Git diff”）
  • Prompts：预定义的提示模板

三、ggml 项目生态一览

3.1 ggml 是什么

ggml（GPT-Generated Model Language）是一个底层 C/C++ 张量计算库，专为在普通硬件上高效运行 ML 模型设计。它是 llama.cpp、whisper.cpp 等项目的共同基座。

ggml-org 在 GitHub 上有 22 个仓库，覆盖从底层库到上层应用的完整生态。

3.6 GGUF 格式

ggml 还定义了 GGUF（GPT-Generated Unified Format）模型文件格式，已成为开源 LLM 量化的事实标准。几乎所有主流开源模型都有社区提供的 GGUF 版本。

GGUF 支持的量化级别：

F32 32-bit 1×
F16 / BF16 16-bit 2×
Q8_0 8.5-bit ~2×
Q6_K 6-bit ~2.5×
Q5_K_M 5.5-bit ~3×
Q4_K_M 4.5-bit ~4×
Q3_K_M 3.5-bit ~5×
Q2_K_S 2.5-bit ~6×
TQ2_0 1.6-bit ~10×（三值）

四、llama-server Agent 参数全解

4.1 Agent 功能概述

llama-server 从 2026 年起内置了 Agent 模式，包含一组内置工具（文件操作、Shell 命令、搜索等），让 LLM 可以直接操作本地系统。这是除了通过 MCP 连接外部工具之外的另一种 Agent 实现方式。

⚠️ 安全警告：Agent 工具可以执行 Shell 命令和文件写入，不要在不可信环境中启用。

4.2 Agent 核心参数

参数 说明
-ag, --agent / --no-agent 启用 CORS 代理和所有内置工具（默认禁用）
--tools TOOL1,TOOL2,... 启用指定内置工具；all 启用全部
--ui-mcp-proxy / --no-ui-mcp-proxy 启用 Web UI 的 MCP CORS 代理（实验性，默认禁用）

--mmproj-offload 投影 GPU 卸载（默认开启）
--image-min-tokens N 图像最小 token
--image-max-tokens N 图像最大 token
--mtmd-batch-max-tokens N 图像编码批大小（默认 1024）
